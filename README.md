Overview

This project analyzes student performance data to identify key factors influencing academic results. The analysis explores how gender, lunch type, and test preparation affect performance, along with overall score distribution and relationships between variables.

Objectives

Understand overall score distribution among students,
Analyze performance differences based on gender,
Evaluate the impact of lunch type on student performance,
Assess the effect of test preparation on scores,
Identify relationships between variables using correlation analysis,
Visualize key insights using statistical plots and heatmaps

Key Analysis Performed

1. Score Distribution

Analysis of how student scores are distributed to identify patterns such as clustering, skewness, and performance spread.

2. Gender Performance Analysis

Comparison of average scores between male and female students to determine performance differences.

3. Lunch Impact Analysis

Evaluation of how lunch type (standard vs free/reduced) affects student academic performance.

4. Test Preparation Impact

Analysis of how completing a test preparation course influences student scores.

5. Education Impact Analysis

Study of how different educational background factors influence student performance outcomes.

6. Correlation Heatmap

A correlation matrix is used to measure relationships between numerical variables and identify strong or weak associations.
